# RPC-based script

## Introduction
This add-on is based on the corresponding sample script located in the sample-scripts folder.
Add-ons are customizable Cisco Crosswork Planning components that allow users to run Python
executable scripts like any other tools and view the results, reports or generated output 
plan files from the Cisco Crosswork Planning Design UI. 


## Import and enable the add-on
Download the `*.tar` file in this folder and import it from the Cisco Crosswork Planning Design UI > Manage add-ons option.
After the add-on is imported, proceed to enable it from the same page.
For detailed instructions, please visit the oficial Cisco Crosswork Planning Design User Guide.


## How to use in CP Design UI
1. From the **Network Design UI**, select **Actions > Add-ons > Import COE Model**.
2. In the **Options** page, complete the settings and select **Next**. Note: Create Demands, Copy Template and Copy Inventory are optional.
<img src="/sample-addons/Import-COE-Model/img/addon_options.png" alt="Agent UI" width=800>
3. In the **Run Settings** page, modify or keep default settings and then select **Submit**.
